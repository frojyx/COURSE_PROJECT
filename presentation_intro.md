# Вступление к защите проекта "NewMusicCatalog"

Добрый день!

Моя работа посвящена разработке настольного приложения для управления музыкальным каталогом — **NewMusicCatalog**.

## Цель и задачи проекта

**Цель проекта** — создание удобного настольного приложения для управления коллекцией музыкальных треков с поддержкой расширенного поиска, сортировки и интеграции с внешними сервисами.

**Основные задачи:**
- Разработка модульной архитектуры приложения
- Реализация функционала управления треками
- Интеграция с Yandex Music API
- Поддержка работы с MP3 файлами и их метаданными
- Создание интуитивно понятного пользовательского интерфейса

## Технологический стек

Проект реализован на **C++23** с использованием фреймворка **Qt6** для графического интерфейса и сетевых операций. Для сборки используется **CMake**. Качество кода контролируется с помощью SonarCloud.

## Структура проекта

Архитектура построена на принципах ООП с модульной организацией:
- **Core** — основная бизнес-логика
- **Integration** — интеграция с внешними API
- **File Operations** — работа с файлами
- **MP3** — обработка MP3 файлов
- **UI** — пользовательский интерфейс

## Ключевые достижения

- Полнофункциональная система управления музыкальным каталогом
- Интеграция с Yandex Music для импорта треков
- Расширенный поиск с множественными фильтрами
- Работа с MP3 метаданными (ID3v1, ID3v2)
- Применение принципов чистой архитектуры
